# PWA Icons

Please add the following icon files to this directory:

- `icon-192x192.png` - 192x192px icon for PWA
- `icon-512x512.png` - 512x512px icon for PWA

You can generate these icons from your logo using online tools like:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/

## Recommended Specifications

### icon-192x192.png
- Size: 192 x 192 pixels
- Format: PNG
- Purpose: Maskable, Any

### icon-512x512.png
- Size: 512 x 512 pixels
- Format: PNG
- Purpose: Maskable, Any

## Tips
- Use simple, recognizable designs
- Ensure good contrast
- Test on different backgrounds (light/dark mode)
- Make sure the icon looks good when cropped to a circle (maskable)
